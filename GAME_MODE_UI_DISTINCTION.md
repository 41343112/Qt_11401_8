# 遊戲模式 UI 區分改進

## 概述
本次更新優化了遊戲模式選擇的 UI 介面，明確區分**本機對弈**（離線模式）和**連線對弈**（線上模式）。

## 變更內容

### 1. 視覺分組
- 將「雙人」和「電腦」按鈕歸類為「本機對弈」
- 將「線上」按鈕單獨歸類為「連線對弈」

### 2. 新增分類標籤
- **💻 本機對弈**：標示不需要網路連線的離線模式
  - 👥 雙人：本地雙人對戰
  - 🤖 電腦：與 AI 對戰
- **🌐 連線對弈**：標示需要網路連線的線上模式
  - 🌐 線上：透過網路與遠端玩家對戰

### 3. 視覺效果改進
- **本機對弈標籤**：使用綠色（霓虹綠色）強調離線特性
- **連線對弈標籤**：使用粉色（霓虹粉色）強調線上特性
- **線上按鈕樣式**：更新為粉色主題，與連線對弈標籤保持一致
- **按鈕布局**：線上按鈕與上方本機按鈕保持對齊

## 使用者體驗改善

### 改進前
三個按鈕並排顯示，沒有明確的分類：
```
👥 雙人  🤖 電腦  🌐 線上
```

### 改進後
明確分為兩個區塊，清楚標示離線和線上模式：
```
⚔ 對弈模式:

💻 本機對弈
👥 雙人  🤖 電腦

🌐 連線對弈
🌐 線上
```

## 技術細節

### 修改的檔案
- `src/qt_chess.cpp`：更新 UI 布局和樣式

### 主要變更
1. 新增 `localModeLabel`：顯示「💻 本機對弈」
2. 新增 `localModeButtonsLayout`：包含雙人和電腦按鈕
3. 新增 `onlineModeLabel`：顯示「🌐 連線對弈」
4. 新增 `onlineModeButtonLayout`：包含線上按鈕
5. 新增 `onlineModeStyle`：線上模式專用的粉色主題樣式

### 相容性
- 所有現有功能保持不變
- 按鈕點擊行為完全相同
- 不影響遊戲邏輯或網路功能

## 設計理念

### 清晰性
- 使用者可以立即識別哪些模式需要網路連線
- 降低使用者誤選不適合的模式的機會

### 一致性
- 標籤顏色與按鈕樣式協調一致
- 保持現代科技風格的整體設計語言

### 可用性
- 分組使介面更有組織性
- 減少認知負擔，提升使用體驗
