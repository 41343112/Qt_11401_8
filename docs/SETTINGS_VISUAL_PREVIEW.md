## 響應式佈局 (Responsive Layout)

### 小視窗模式 (Small Window Mode)
- 最小視窗大小: 814 x 480 像素
- 按鈕自動調整以適應視窗大小
- 保持最小寬度 250px 和最小高度 40px

### 大視窗模式 (Large Window Mode)
- 推薦視窗大小: 900 x 660 像素
- 所有元素居中對齊
- 保持舒適的間距和邊距

## 使用者互動流程 (User Interaction Flow)

1. **進入設定面板**
   - 使用者在主選單點擊「⚙️ 設定」
   - 主選單淡出，設定面板淡入
   - 所有設定選項顯示

2. **選擇設定項目**
   - 滑鼠懸停在按鈕上時顯示懸停效果
   - 點擊按鈕打開對應的設定對話框
   - 對話框以模態方式顯示

3. **返回主選單**
   - 點擊「← 返回主選單」按鈕
   - 設定面板淡出，主選單淡入
   - 恢復主選單狀態

## 總結 (Summary)

設定面板提供了一個直觀、美觀且易於使用的界面，讓使用者可以輕鬆訪問和配置遊戲的各種設定。通過與主選單保持一致的設計風格，確保了整體使用者體驗的連貫性和專業性。
